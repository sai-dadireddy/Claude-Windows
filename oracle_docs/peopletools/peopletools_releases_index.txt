## Upgrade Path (8.62)

From PeopleTools version:
- 8.54 or higher → 8.62 (direct upgrade supported)
- 8.5x versions → 8.62 (requires Change Assistant)
- Below 8.54 → Must upgrade to 8.54+ first

Databases supported:
- Oracle Database
- Microsoft SQL Server (certified for 8.62.03+)
- IBM DB2 for z/OS (certified for cloud customers)
