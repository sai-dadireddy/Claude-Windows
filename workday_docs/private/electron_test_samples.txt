DESCRIPTION:

Organizations' for any Cost Center Company Hierarchy (if applicable). This test

14. enter search box as "Find Organizations"
15. wait for search results
16. click search result containing "Find Organizations"
17. wait for page to load
18. select organization type filter "Company"
19. click "OK" to search
20. wait for results to load
21. verify company count matches extract report
22. screenshot as "HCM-1-0010_complete.png"

VERIFICATION:
- [ ] All companies are present in the extract
- [ ] Company data is complete (no missing Reference IDs or names)
- [ ] Cost Center Company Hierarchy includes expected organizations
- [ ] Company count matches between extract and search results
- [ ] Data quality is accurate and matches source system

TEST ID: USP-1-0010
FUNCTIONAL AREA: Payroll for the United States
TEST NAME: Review Period Schedule Configuration
WORKDAY ROLE: Payroll Administrator
EST. DURATION: 5 minutes
================================================================================

DESCRIPTION:
Verify that all expected period schedules are present in the tenant, dates are
correct (period start, period end, payment date, and Accounting Date Override)
and that forward accrual settings are as expected. Verify that period schedules
are linked to Absence/Time Tracking as applicable. Ensure that the correct
number of periods are in place for each year. If applicable, add Continuous
Payroll Calculation configuration.

PREREQUISITES:
- User logged in with Payroll Administrator permissions
- Period schedules have been configured in the tenant
- Fiscal year and calendar configuration is complete
- Access to payroll configuration domain

ELECTRON STEPS:
1. enter search box as "View Period Schedule"
2. wait for search results
3. click search result containing "View Period Schedule"
4. wait for page to load
5. verify period schedule selection page displays
6. click dropdown "Select Period Schedule"
7. select first period schedule from list
8. click "OK" to view schedule
9. wait for schedule details to load
10. verify schedule details section displays:
    - Period Schedule Name
    - Frequency (Weekly, Bi-Weekly, Semi-Monthly, Monthly)
    - Number of periods per year
11. scroll to period dates table
12. verify column headers: Period, Period Start, Period End, Payment Date, Accounting Date Override
13. for each period row:
    - verify Period Start Date is present
    - verify Period End Date is present
    - verify Payment Date is present
    - verify dates are in logical sequence
14. click "Forward Accrual Settings" tab
15. verify forward accrual configuration is as expected
16. screenshot as "USP-1-0010_period_schedule.png"
17. click "Time Tracking" tab (if applicable)
18. verify time tracking linkage is configured
19. click "Absence" tab (if applicable)
20. verify absence linkage is configured
21. screenshot as "USP-1-0010_complete.png"

VERIFICATION:
- [ ] All expected period schedules are present in the tenant
- [ ] Period start dates, end dates, and payment dates are accurate
- [ ] Accounting Date Override is configured correctly (if applicable)
- [ ] Forward accrual settings match requirements
- [ ] Period schedules are linked to Absence/Time Tracking as needed
- [ ] Correct number of periods exist for each fiscal year
- [ ] Continuous Payroll Calculation is configured (if applicable)
- [ ] Period dates follow logical sequence without gaps or overlaps

API ALTERNATIVE:
- SOAP Service: Payroll
  Operation: Get_Period_Schedules
  Request Parameters:
    - Period_Schedule_Reference (optional for specific schedule)
    - Include_Period_Schedule_Data: true

  Sample SOAP Request:
  <bsvc:Get_Period_Schedules_Request>
    <bsvc:Request_Criteria>
      <bsvc:Include_Period_Schedule_Data>true</bsvc:Include_Period_Schedule_Data>
    </bsvc:Request_Criteria>
  </bsvc:Get_Period_Schedules_Request>

- REST API: Payroll API (if available in your tenant version)
  Endpoint: /api/v1/{tenant}/periodSchedules
  Method: GET
  Parameters: ?limit=100&offset=0

RELATED WSDL OPERATIONS:
- Get_Payroll_Calendar: Retrieve payroll calendar configuration
- Get_Time_Off_Plan: Review time off plans linked to period schedules
- Submit_Payroll_Input: Submit payroll for a specific period

NOTES:
- Period schedules cannot be deleted once payroll has been run
- Changes to period dates may require payroll recalculation
- Verify period schedule assignment to employee populations
