================================================================================
OVERVIEW
================================================================================

Settlement Run in Workday Canada is the process of finalizing payroll and
generating payments to employees. This includes:
- Locking payroll results (no further changes)
- Generating payment files (EFT/Direct Deposit)
- Creating checks for non-EFT employees
- Generating GL journal entries
- Processing employee wage payments

Settlement is a critical step that commits payroll results and initiates
the payment process to employees.

Business Process: Create Settlement Run
Functional Area: Payroll for Canada
Initiator: Payroll Settlement Specialist
Typical Duration: 15-45 minutes

================================================================================
KEY COMPONENTS
================================================================================

1. COMPANY HIERARCHY
   - Parent company selection
   - Child companies included
   - Multi-company settlement support
   - Quebec-specific company handling

2. PAY GROUP SELECTION
   - All Available Pay Groups option
   - Specific pay group selection
   - Pay Group Exceptions
   - Multiple pay frequencies

3. PAYMENT METHODS
   - EFT (Electronic Funds Transfer)
   - Direct Deposit (Canadian banks)
   - Cheque (check) printing
   - Wire transfers

4. BANK ROUTING
   - Bank account assignments
   - Routing rules configuration
   - CAD currency handling
   - Multi-bank support

5. GL POSTING
   - Payroll journal entries
   - Cost center allocation
   - Account mapping
   - Suspense account handling

================================================================================
NAVIGATION & TASKS
================================================================================

Create Settlement Run:
1. Search: "Create Settlement Run"
2. Select company hierarchy (for multi-company)
3. Select filters (pay groups, exceptions)
4. Verify settlement parameters
5. Submit settlement run

View Settlement Run:
1. Search: "View Settlement Run"
2. Select settlement run ID
3. Review settlement details and status

Cancel Settlement Run:
1. Search: "Cancel Settlement Run"
2. Select settlement run
3. Confirm cancellation (before bank file submission)

================================================================================
PREREQUISITES
================================================================================

Before Settlement:
- All pay calculations complete
- No pay results in error status
- Bank accounts assigned to employees
- Bank routing rules configured
- GL accounts mapped
- Payroll approval completed

================================================================================
ERROR HANDLING
================================================================================

Common Errors:
- "No complete pay results": Run pay calculation first
- "Missing bank account": Assign bank to employee
- "Bank routing error": Check bank routing rules
- "GL account not found": Map payroll accounts
- "Suspense account posting": Fix accounting configuration

Resolution Steps:
1. Review error details
2. Fix underlying configuration
3. Recalculate affected pay results
4. Re-attempt settlement

- Verify payroll register
