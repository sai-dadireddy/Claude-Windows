================================================================================
OVERVIEW
================================================================================

Pay Run Groups in Workday Canada organize pay groups for consolidated payroll
processing. They determine:
- Which pay groups are processed together
- Processing sequence and dependencies
- Multi-company payroll coordination
- Settlement run groupings

Business Process: View/Configure Pay Run Group
Functional Area: Payroll for Canada
Initiator: Payroll Administrator
Typical Duration: 10-20 minutes

================================================================================
KEY COMPONENTS
================================================================================

1. PAY RUN GROUP ELEMENTS
   - Pay Run Group Name
   - Pay Run Group ID
   - Included Pay Groups
   - Processing Order
   - Company assignments

2. PAY GROUP RELATIONSHIPS
   - Primary pay groups
   - Secondary/subordinate groups
   - Cross-company groupings
   - Period schedule alignment

3. PROCESSING SETTINGS
   - Calculation sequence
   - Settlement run association
   - Approval workflow alignment
   - Batch processing configuration

================================================================================
NAVIGATION & TASKS
================================================================================

View Pay Run Group:
1. Search: "View Pay Run Group"
2. Select pay run group
3. Review included pay groups

Configure Pay Run Group:
1. Search: "Configure Pay Run Group"
2. Create or modify pay run group
3. Assign pay groups

================================================================================
VALIDATION CHECKLIST
================================================================================

- All expected pay run groups exist
- Pay groups correctly assigned
- Processing order appropriate
- No gaps in pay group coverage
- Settlement associations correct
