KEY WORKDAY TASKS & NAVIGATION
================================================================================

## 1. CREATE JOB REQUISITION
Task: Create Job Requisition
Navigation: Search "Create Job Requisition" | Recruiting Hub > Create Requisition
WSDL Operation: Create_Job_Requisition

Required Fields:
- Supervisory Organization (org hierarchy)
- Position (existing or create new)
- Job Title
- Job Profile
- Worker Type (Employee/Contingent Worker)
- Time Type (Full-time/Part-time)
- Location
- Primary Recruiting Contact
- Hiring Manager

Steps:
1. Search "Create Job Requisition"
2. Select Supervisory Organization
3. Select or Create Position
4. Enter Job Details (Title, Profile, Location)
5. Set Recruiting Details (Contact, Hiring Manager)
6. Configure Posting Details (if posting immediately)
7. Submit for Approval

Verification:
- Job Requisition ID generated (JR-XXXXX format)
- Status shows "Open" or "In Approval"
- Position linked to requisition

--------------------------------------------------------------------------------

## 2. POST JOB
Task: Post Job
Navigation: Job Requisition > Related Actions > Post Job
WSDL Operation: Post_Job

Required Fields:
- Job Requisition Reference
- Career Site(s) (Internal/External)
- Posting Start Date
- Posting End Date (optional)

Steps:
1. Navigate to Job Requisition
2. Click Related Actions > Post Job
3. Select Career Site(s)
4. Set Posting Dates
5. Review Job Posting Content
6. Submit

Verification:
- Job appears on selected career site(s)
- Posting status shows "Active"
- Job Posting reference created

--------------------------------------------------------------------------------

## 3. CREATE PROSPECT
Task: Create Prospect
Navigation: Search "Create Prospect" | Recruiting > Create Prospect
REST API: POST /prospects

Required Fields:
- First Name
- Last Name
- Email Address
- Source (how candidate was found)
- Recruiting Region (optional)

Steps:
1. Search "Create Prospect"
2. Enter Personal Information (Name, Email)
3. Add Contact Information (Phone, Address)
4. Select Recruiting Source
5. Add Resume/Attachments (optional)
6. Submit

Verification:
- Prospect record created
- Prospect ID generated
- Appears in talent pipeline

--------------------------------------------------------------------------------

## 4. EXTERNAL CANDIDATE JOB APPLICATION
Task: External Candidate Applies for Job
Navigation: Career Site > Job Posting > Apply
REST API: POST /jobPostings/{ID}/applications

Steps (External Candidate):
1. Access External Career Site
2. Search/Browse Job Postings
3. Select Job and Click "Apply"
4. Create Candidate Account (if new)
5. Complete Application Form
6. Upload Resume/Documents
7. Answer Questionnaire (if configured)
8. Submit Application

Verification:
- Application submitted confirmation
- Application appears in Job Requisition
- Candidate receives confirmation email

--------------------------------------------------------------------------------

## 5. INTERNAL CANDIDATE JOB APPLICATION
Task: Internal Candidate Applies for Job
Navigation: Internal Career Site > Career Hub > Find Jobs
Role: Any Worker with internal mobility access

Steps:
1. Navigate to Internal Career Site
2. Search for Open Positions
3. Select Job and Click "Apply"
4. Review Pre-populated Information
5. Complete Additional Questions
6. Submit Application

Verification:
- Application shows "Internal Candidate" flag
- Manager notification triggered (if configured)
- Application in candidate pool

--------------------------------------------------------------------------------

## 6. REVIEW CANDIDATE
Task: Review Candidate
Navigation: Job Requisition > Candidates > Review
WSDL Operation: Get_Candidates

Steps:
1. Navigate to Job Requisition
2. Click Candidates tab
3. Select Candidate to Review
4. Review Application Details
5. View Resume/Attachments
6. Add Recruiting Notes (optional)
7. Move to Next Stage or Disposition

Verification:
- Candidate stage updated
- Review timestamp recorded
- Notes saved to candidate record

--------------------------------------------------------------------------------

## 7. SCREEN CANDIDATE
Task: Screen Candidate
Navigation: Job Application > Move to Screen Stage
Business Process: Job Application > Screen step

Steps:
1. Select Candidate from Requisition
2. Click "Move Candidate" or Related Actions
3. Select "Screen" stage
4. Complete Screening Questions
5. Add Evaluation/Notes
6. Submit

Verification:
- Candidate moved to Screen stage
- Screening data captured
- Next step in workflow triggered

--------------------------------------------------------------------------------

## 8. ASSESS CANDIDATE
Task: Assess Candidate
Navigation: Job Application > Assess Candidate
WSDL Operation: Assess_Candidate

Steps:
1. Select Candidate in Interview stage
2. Click "Assess Candidate"
3. Complete Assessment Form
4. Rate Skills/Competencies
5. Add Overall Assessment Notes
6. Submit

Verification:
- Assessment recorded
- Stage updated if auto-advance configured
- Assessment visible to recruiting team

--------------------------------------------------------------------------------

## 9. SCHEDULE INTERVIEW
Task: Schedule Interview
Navigation: Job Application > Schedule Interview
WSDL Operations: Put_Interview, Get_Interviews

Required Fields:
- Interview Type
- Date/Time
- Location (or Virtual Meeting Link)
- Interviewer(s)
- Interview Duration

Steps:
1. Select Candidate
2. Click "Schedule Interview"
3. Select Interview Type
4. Set Date, Time, Duration
5. Add Interviewers
6. Set Location/Virtual Link
7. Add Interview Instructions
8. Submit/Send Invites

Verification:
- Interview scheduled
- Calendar invites sent to participants
- Candidate notification sent

WSDL Operation: Put_Interview_Feedback

Steps:
1. Access Interview Event
2. Review Candidate Information
3. Conduct Interview
4. Complete Interview Feedback Form
5. Rate Candidate (Overall Rating)
6. Add Comments/Notes
7. Submit Feedback

Verification:

- Rating captured

## 11. SUBMIT OFFER
Task: Submit Offer
Navigation: Job Application > Submit Offer | Related Actions > Offer
WSDL Operation: Offer

Required Fields:
- Compensation Details
- Start Date
- Position Details
- Offer Letter Template (optional)

Steps:
1. Select Candidate (Ready for Offer stage)
2. Click "Submit Offer" or Related Actions > Offer
3. Configure Compensation
4. Set Proposed Start Date
5. Review Position Details
6. Select Offer Letter Template
7. Preview Offer Letter
8. Submit for Approval (if BP configured)

Verification:
- Offer created
- Pending approval (if configured)
- Candidate notified when approved

--------------------------------------------------------------------------------

## 12. PROCESS BACKGROUND CHECK
Task: Initiate Background Check
Navigation: Job Application > Background Check | Related Actions
WSDL Operations: Put_Background_Check, Get_Background_Check

Steps:
1. Select Candidate (after offer acceptance)
2. Click "Initiate Background Check"
3. Select Background Check Package
4. Review Candidate Consent
5. Submit to Provider

Verification:
- Background check initiated
- Status shows "In Progress"
- Provider receives request

--------------------------------------------------------------------------------

## 13. READY TO HIRE
Task: Move to Ready to Hire
Navigation: Job Application > Move to Ready to Hire
Business Process: Job Application completion step

Steps:
1. Verify all stages complete
2. Background check cleared (if required)
3. Click "Ready to Hire"
4. Confirm Start Date
5. Submit

Verification:
- Candidate status "Ready to Hire"
- Hire process can be initiated
- Pre-hire record created

WSDL Operation: Close_Job_Requisition
