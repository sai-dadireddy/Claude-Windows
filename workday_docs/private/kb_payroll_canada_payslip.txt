================================================================================
OVERVIEW
================================================================================

Payslip configuration in Workday Canada controls how employee payslips are
displayed, including:
- Earnings display (regular, overtime, bonuses)
- Deductions (CPP, EI, income tax, voluntary deductions)
- Employer contributions
- Year-to-date (YTD) totals
- Pay component grouping
- French/English language support

The payslip is the online view shown to employees and may differ from printed
check layouts. PDF check print layouts can be viewed separately.

Business Process: Review Payslip Configuration / View Payslip
Functional Area: Payroll for Canada
Initiator: Payroll Partner / Payroll Administrator
Typical Duration: 15-30 minutes

================================================================================
KEY COMPONENTS
================================================================================

1. PAY COMPONENT GROUPS
   - Current Earnings (salary, hourly, OT, bonus)
   - Current Deductions (CPP, EI, Federal Tax, Provincial Tax)
   - Current Employer Contributions
   - YTD Earnings
   - YTD Deductions
   - Net Pay section

2. CANADIAN-SPECIFIC ELEMENTS
   - CPP/QPP contributions (employee and employer)
   - EI premiums (employee and employer)
   - Federal income tax
   - Provincial income tax
   - Quebec-specific deductions (QPP, QPIP)
   - RRSP contributions
   - Group benefits deductions

3. PAYSLIP SECTIONS
   - Employee information
   - Pay period details
   - Current period earnings
   - Current period deductions
   - Current period employer costs
   - Net pay
   - Year-to-date summary
   - Leave balances

================================================================================
NAVIGATION & TASKS
================================================================================

Review Payslip Configuration:
1. Search: "Maintain Payslip Configuration"
2. Select applicable pay group or company
3. Review pay component group assignments
4. Verify ordering and display settings

View Employee Payslip:
1. Search: "Pay Calculation Results for Worker"
2. Enter worker ID or name
3. Select pay period
4. Click Related Actions > Payroll Result > View Payslip
5. Review payslip display

Generate Payslip PDF:
1. From pay result, select Related Actions
2. Click "Print Payslip"
3. Download PDF for review

================================================================================
VALIDATION CHECKLIST
================================================================================

Configuration:
- All pay component groups configured
- Correct ordering (earnings before deductions)
- YTD sections visible
- Net pay calculated correctly
- Canadian statutory deductions displayed
- Employer contributions section complete

Display:
- Employee name and ID correct
- Pay period dates accurate
- Check date correct
- All earnings visible
- CPP/QPP contributions shown
- EI premiums displayed
- Federal tax withheld
- Provincial tax withheld
- RRSP contributions (if applicable)
- Net pay correct

French Requirements (Quebec):
- French language option available
- Terminology correct in French
- All amounts formatted properly

================================================================================
COMMON ISSUES
================================================================================

- Missing pay component groups: Update Payslip Configuration
- Incorrect ordering: Reorder pay component groups
- Missing YTD: Ensure YTD sections added to configuration
- French not displaying: Check language settings for Quebec employees
- Net pay mismatch: Review calculation rules
