================================================================================
OVERVIEW
================================================================================

Payroll Security in Workday Canada controls access to:
- Payroll data (view/edit permissions)
- Payroll processing tasks
- Settlement operations
- Tax configuration
- Payroll reports

Security is managed through Security Groups with user assignments.

Business Process: Review/Assign Payroll Security Groups
Functional Area: Payroll for Canada / Security
Initiator: Security Administrator
Typical Duration: 15-30 minutes

================================================================================
KEY SECURITY GROUPS
================================================================================

1. PAYROLL ADMINISTRATOR
   - Full access to payroll configuration
   - Tax setup permissions
   - Pay group management
   - Period schedule management

2. PAYROLL PARTNER
   - Payroll processing permissions
   - Results review and approval
   - Input corrections
   - Report access

3. PAYROLL SETTLEMENT SPECIALIST
   - Settlement run creation
   - Payment processing
   - Bank file generation
   - Settlement review

4. PAYROLL ANALYST
   - Report access (read-only)
   - Payroll data analysis
   - No processing permissions

5. HR PARTNER (LIMITED)
   - View employee pay data
   - Limited payroll visibility
   - No processing access

NAVIGATION & TASKS
================================================================================

View Security Group Members:
1. Search: "View Security Group Members"

Assign Security Groups:
1. Search: "Assign Security Groups"

Audit Security Assignments:
1. Search: "Security Group Membership Report"
2. Filter by payroll groups
3. Export for review
