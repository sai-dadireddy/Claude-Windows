## File Naming Patterns Supported

The update script handles two filename patterns:

1. **Standard:** FACC-X-XXXX_Task Name.txt
   - Example: FACC-1-0130_Edit Tenant Setup Financials.txt

2. **Sub-numbered:** FACC-X-XXXX-XX_Task Name.txt
   - Example: FACC-4-0240-01_Intercompany Supplier Settlement.txt

## Generated Electron Step Patterns

### For "Confirm/Verify" Tasks
1. enter search box as "{Task Name}"
2. wait for search results
3. click search result containing "{Task Name}"
4. wait for page to load
5. verify data displayed correctly
6. check required fields are populated
7. screenshot as "{TEST_ID}_complete.png"

### For "Edit" Tasks
1. enter search box as "{Task Name}"
2. wait for search results
3. click search result containing "{Task Name}"
4. wait for page to load
5. modify required fields
6. click button "OK"
7. screenshot as "{TEST_ID}_complete.png"

### For "Audit" Tasks
1. enter search box as "{Task Name}"
2. wait for search results
3. click search result containing "{Task Name}"
4. wait for page to load
5. review displayed information
6. verify against expected values
7. screenshot as "{TEST_ID}_complete.png"
