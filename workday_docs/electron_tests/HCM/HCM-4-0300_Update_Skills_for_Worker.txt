DESCRIPTION:
Navigate to the Career for Worker Profile tab and edit Skills. Verify suggested skills are available to add to the worker record.

WORKDAY ROLE: HR Administrator
EST. DURATION: 10 minutes

PREREQUISITES:
- User logged in with HR Administrator permissions
- Required data available in tenant
- Worker ID for skills update

ELECTRON STEPS:
1. enter search box as "{worker_name}"
2. wait for search results
3. click worker result containing "{worker_name}"
4. wait for worker profile to load
5. click tab "Career"
6. wait for page to load
7. scroll to "Skills"
8. click button "Edit" in Skills section
9. wait for skills editor to load
10. click button "Add" or "Add Skill"
11. enter Skill field as "Python Programming"
12. wait for skill search results
13. click skill result "Python Programming"
14. select Proficiency Level as "Advanced"
15. enter Years of Experience as "5"
16. scroll to "Suggested Skills" section
17. verify suggested skills are displayed
18. click button "Add" next to first suggested skill
19. select Proficiency Level as "Intermediate"
20. click button "Submit" or "Save"
21. wait for confirmation
22. verify message contains "successfully updated"
23. screenshot as "HCM-4-0300_complete.png"
