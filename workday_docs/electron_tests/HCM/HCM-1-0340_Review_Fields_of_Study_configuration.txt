1. enter search box as "All Fields of Study"
2. wait for search results
3. click search result containing "All Fields of Study"
4. wait for page to load
5. screenshot as "HCM-1-0340_report_view.png"

14. select filter as "Yes" to show only active fields
15. wait for filter to apply
16. screenshot as "HCM-1-0340_active_filtered.png"
17. verify expected fields of study appear in the list
18. click link "Export to Excel" if export verification needed
19. screenshot as "HCM-1-0340_complete.png"
