SCHOOL CONFIGURATION ITEMS:
- School ID: Unique identifier (often IPEDS code for US)
- School Name: Official institution name
- School Type: University, College, Community College, High School, etc.
- Country: Country where school is located
- Region/State: State or province
- Status: Active/Inactive
- External ID: Integration reference

FIELD REFERENCE:
- School Name: Official name of educational institution
- School Type: Dropdown (University, College, etc.)
- Country: Reference to country
- IPEDS Code: US Department of Education identifier
- External ID: Integration system reference
- Inactive: Checkbox for obsolete schools

COMMON SCHOOL TYPES:
- University: 4-year degree-granting institution
- College: 2-4 year institution
- Community College: 2-year institution
- High School: Secondary education
- Vocational School: Trade/technical training
- Online: Distance learning institution

ERROR HANDLING:
- "School not found": Verify school name spelling
- "Duplicate school": School may already exist
- "Access denied": User needs HR administrator access
- "Invalid country": Check country reference data
