TASK/STEP: Create Supplier Contract
NOTE: Task inferred from scenario description - original Excel Task/Step was null

DESCRIPTION:
Create a Supplier Contract for an Operating Lease using the ROU Asset Operating
Expense accounting method. This scenario configures the lease to report under
only one accounting standard (typically US GAAP ASC 842).

The ROU Asset Operating Expense method recognizes the right-of-use asset and
lease liability on the balance sheet, with lease expense recorded as an operating
expense over the lease term.

PREREQUISITES:
- User logged in with Lease Accounting Administrator permissions
- Lease Book Code Configuration exists (see LA-1-0070)
- Supplier Contract Types configured for leases (see LA-1-0040)
- Active Supplier/Lessor exists in system
- Asset Book Rules configured (see LA-1-0060)
