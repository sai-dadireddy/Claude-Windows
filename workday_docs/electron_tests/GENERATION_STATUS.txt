================================================================================
GENERATED TEST FILE FORMAT
================================================================================

Each .txt file contains:

1. METADATA
   - Scenario ID (e.g., TLO-1-0010)
   - Module name
   - Test type
   - Workday role required
   - CONFIDENCE SCORE (0.0-1.0)

2. CONFIDENCE SCORING
   >= 0.70  HIGH      - Ready for implementation
   0.40-0.69 MEDIUM   - Needs SME review
   < 0.40   LOW       - Manual investigation required

3. SCENARIO DETAILS
   - Description
   - Task/Step
   - Expected Result

4. RAG QUERY RESULTS
   - API/WSDL information from knowledge base
   - Relevant documentation excerpts

5. ELECTRON TEST TEMPLATE
   - Playwright/JavaScript structure
   - Prerequisites
   - Test steps (TODO placeholders)

6. API INTEGRATION POINTS
   - REST/SOAP alternatives
   - Endpoint information when available

7. DATA REQUIREMENTS
   - Scope type
   - Country code
   - Industry

8. TEST STATUS CHECKLIST
   - [ ] Test implemented
   - [ ] Test executed
   - [ ] Results verified
   - [ ] Edge cases covered
