================================================================================
WHAT EACH GENERATOR DOES
================================================================================

STEP 1: Read Excel Sheet
   - Loads scenarios from specific sheet (e.g., Expenses)
   - Validates required columns: Scenario ID, Scenario Name, Task/Step, Expected Result

STEP 2: Query RAG for Each Scenario
   - Searches Workday knowledge base using "Task / Step" field
   - Queries 117 documents (PDFs, WSDLs, KB articles)
   - Extracts relevant automation steps

STEP 3: Calculate Confidence Score
   - 7.0-10.0 (HIGH): Complete automation possible
   - 4.0-6.9 (MEDIUM): Partial automation, needs review
   - < 4.0 (LOW): Manual test required

STEP 4: Generate Electron Steps
   Creates Electron-compatible commands:
   1. enter search box as "Create Expense Report"
   2. wait for search results
   3. click search result
   4. wait for page load
   5. enter expense details
   6. click "Submit" button
   7. screenshot as "EXP-001_complete.png"

STEP 5: Write Test File
   - Individual .txt file per scenario
   - Includes: metadata, Electron steps, verification, RAG insights
