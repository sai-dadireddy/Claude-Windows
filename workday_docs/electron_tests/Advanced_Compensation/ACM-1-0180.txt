DESCRIPTION:
Review the validation rules for Merit, Bonus, Stock, Allowance, Commission and Organization
and confirm they are all configured as per requirements. NOTE: New Validation Rules cannot
be added once a compensation review process has begun; only existing rules can be edited.

TASK/STEP:
Search bar: "comp valid " Click on Maintain Compensation Review Validations. Click on each one listed.

PREREQUISITES:
- User logged in with Compensation Administrator permissions
- Required data: Validation rules to review

ELECTRON STEPS:
1. enter search box as "Maintain Compensation Review Validations"
2. wait for search results
3. click search result containing "Maintain Compensation Review Validations"
4. wait for page to load
5. screenshot as "ACM-1-0180_validations_list.png"
6. verify list of compensation review validations is displayed
7. click link for Merit validation rule
8. wait for validation details to load
9. verify validation criteria is displayed
10. screenshot as "ACM-1-0180_merit_validation.png"
11. click button "Back" or navigate back
12. click link for Bonus validation rule
13. wait for validation details to load
14. verify validation criteria is displayed
15. screenshot as "ACM-1-0180_bonus_validation.png"
16. click button "Back" or navigate back
17. click link for Stock validation rule
18. verify validation criteria is displayed
19. screenshot as "ACM-1-0180_stock_validation.png"
20. screenshot as "ACM-1-0180_complete.png"
