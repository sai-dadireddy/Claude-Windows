DESCRIPTION:
To verify that the Stock plans are configured correctly:
- Targets hidden/overridden
- Applicable profiles
- Eligibility Rules
- Rounding Rule
- Defaults

TASK/STEP:
Search bar: "comp plans". Click on the Stock tab. From the Stock tab, select the applicable Stock Plan(s) to review.

PREREQUISITES:
- User logged in with Compensation Administrator permissions
- Required data: Stock plan names to review

ELECTRON STEPS:
1. enter search box as "Maintain Compensation Plans"
2. wait for search results
3. click search result containing "Maintain Compensation Plans"
4. wait for page to load
5. click tab "Stock"
6. wait for tab content to load
7. screenshot as "ACM-1-0050_stock_plans_list.png"
8. click link for first Stock Plan in list
9. wait for plan details to load
10. verify field "Plan Name" is displayed
11. verify field "Effective Date" is displayed
12. verify section "Eligibility Rules" is displayed
13. verify section "Rounding Rule" is displayed
14. verify section "Grant Type" is displayed
15. verify section "Vesting Schedule" is displayed
16. screenshot as "ACM-1-0050_stock_plan_config.png"
17. scroll to "Defaults" section
18. verify default configuration is displayed
19. screenshot as "ACM-1-0050_complete.png"

VERIFICATION:
- [ ] Stock tab displays list of configured stock plans
- [ ] All configuration fields are visible and populated
- [ ] Eligibility Rules section shows applicable rules
- [ ] Grant Type is assigned
- [ ] Vesting Schedule is configured
- [ ] No error messages displayed
- [ ] Task completed successfully
