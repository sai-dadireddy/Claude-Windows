DESCRIPTION:
To verify that the Merit plans are configured correctly:
- Targets hidden/overridden
- Applicable profiles
- Eligibility Rules
- Statutory Minimum
- Waiting Period
- Compensation Matrix
- Time Proration Rule
- Rounding Rule

TASK/STEP:
Search bar: "comp plans". Click on the Merit tab. From the Merit tab, select the applicable Merit Plan(s) to review.

PREREQUISITES:
- User logged in with Compensation Administrator permissions
- Required data: Merit plan names to review

ELECTRON STEPS:
1. enter search box as "Maintain Compensation Plans"
2. wait for search results
3. click search result containing "Maintain Compensation Plans"
4. wait for page to load
5. click tab "Merit"
6. wait for tab content to load
7. screenshot as "ACM-1-0010_merit_plans_list.png"
8. click link for first Merit Plan in list
9. wait for plan details to load
10. verify field "Plan Name" is displayed
11. verify field "Effective Date" is displayed
12. verify section "Eligibility Rules" is displayed
13. verify section "Time Proration Rule" is displayed
14. verify section "Rounding Rule" is displayed
15. verify section "Compensation Matrix" is displayed
16. screenshot as "ACM-1-0010_merit_plan_config.png"
17. scroll to "Waiting Period" section
18. verify field "Waiting Period" configuration is displayed
19. screenshot as "ACM-1-0010_complete.png"

VERIFICATION:
- [ ] Merit tab displays list of configured merit plans
- [ ] All configuration fields are visible and populated
- [ ] Eligibility Rules section shows applicable rules
- [ ] Compensation Matrix is assigned if applicable
- [ ] No error messages displayed
- [ ] Task completed successfully
