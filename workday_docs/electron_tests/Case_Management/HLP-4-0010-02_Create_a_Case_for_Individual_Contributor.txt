DESCRIPTION:
Verify questionnaires appear as expected for selected case types.

This test validates that when an Individual Contributor selects a Case Type
that has a questionnaire configured, the questionnaire displays correctly
with all expected fields.

PREREQUISITES:
- User logged in with Employee As Self permissions
- At least one Case Type must have a questionnaire attached
- Case Eligibility Rules allow employee to see the Case Type
- Test data: Know which Case Type has questionnaire configured

================================================================================
ELECTRON STEPS:
================================================================================

-- STEP 1: Navigate to Create Case --
1. enter search box as "Create Case"
2. wait for search results
3. click search result containing "Create Case"
4. wait for page to load
5. screenshot as "HLP-4-0010-02_create_case_landing.png"

-- STEP 2: Select Service Category --
6. verify page contains "Service Category" or category tiles
7. click tile for Service Category that has Case Type with questionnaire
8. wait for case types to load
9. screenshot as "HLP-4-0010-02_service_categories.png"

-- STEP 3: Select Case Type with Questionnaire --
10. click tile for Case Type that has questionnaire attached
11. wait for questionnaire to load
12. wait 2 seconds
13. screenshot as "HLP-4-0010-02_questionnaire_loaded.png"

-- STEP 4: Verify Questionnaire Appears --
14. verify page contains questionnaire section
15. verify questionnaire fields are displayed
16. verify required field indicators are present

-- STEP 5: Interact with Questionnaire (validation) --
17. scroll to view all questionnaire fields
18. screenshot as "HLP-4-0010-02_questionnaire_fields.png"

-- STEP 6: Complete Test --
19. click button "Cancel"
20. screenshot as "HLP-4-0010-02_complete.png"

================================================================================

================================================================================

- [ ] Questionnaire displays when Case Type with questionnaire is selected
- [ ] All configured questionnaire questions appear
- [ ] Required fields are clearly marked
- [ ] Questionnaire section is visible and interactive
- [ ] Form fields match expected configuration for Case Type
