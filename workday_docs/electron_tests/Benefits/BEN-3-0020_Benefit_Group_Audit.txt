TASK/STEP: Audit Benefit Groups Configuration

DESCRIPTION:

and employee populations are correct. Ensure each benefit group serves
the intended population with appropriate plan offerings.

PREREQUISITES:
- User logged in with Benefits Administrator role
- Benefit groups configured in system
- Expected benefit group configuration documented
