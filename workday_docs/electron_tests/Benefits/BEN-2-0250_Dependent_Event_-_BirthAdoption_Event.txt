TASK/STEP: Change Benefits > Birth/Adoption Event (Add Dependent to Coverage)

DESCRIPTION:
Process a birth or adoption life event to add new child as dependent on
health, dental, and vision benefit plans. Updates coverage levels from
Employee Only or Employee+Spouse to include children.

PREREQUISITES:
- User logged in with Benefits Administrator or HR Partner role
- Test employee with active medical/dental/vision benefits
- Birth/adoption event date and child information available
- Birth certificate or adoption papers available for upload

ELECTRON STEPS:
1. enter search box as "Change Benefits"
2. wait for search results
3. click search result containing "Change Benefits"
4. wait for page to load
5. enter worker as "{test_employee_id}"
6. wait for worker profile to load
7. select benefit event type as "Life Event"
8. select life event reason as "Birth/Adoption"
9. enter event date as "{birth_adoption_date}"
10. click button "Continue"
11. wait for benefits election page to load
12. scroll to dependents section
13. click button "Add Dependent"
14. enter dependent first name as "{child_first_name}"
15. enter dependent last name as "{child_last_name}"
16. enter date of birth as "{child_dob}"
17. select relationship as "Child"
18. enter social security number as "{child_ssn}"
19. select gender as "{child_gender}"
20. click button "Upload Document"
21. select file "{birth_certificate_path}"
22. click button "Save Dependent"
23. verify new dependent appears in list
24. click link "Medical" in benefits section
25. select coverage level as "Employee + Child(ren)" or "Family"
26. verify new child is included in coverage
27. click button "Elect"
28. click link "Dental" in benefits section
29. select coverage level as "Employee + Child(ren)" or "Family"
30. verify new child is included in coverage
31. click button "Elect"
32. click link "Vision" in benefits section
33. select coverage level as "Employee + Child(ren)" or "Family"
34. verify new child is included in coverage
35. click button "Elect"

37. wait for review page to load

39. verify updated premium costs displayed
40. click button "Submit"

43. screenshot as "BEN-2-0250_complete.png"
