DESCRIPTION:
Draft, Edit & Publish an RFx Event with Suppliers & Stakeholders tagged

TASK / STEP:
1. Navigate to an in progress sourcing Project
2. Click the Sourcing tab
3. Click Create New RFX. Select the template created in the previous test case. The event will add to the project in Draft status.
4. Edit the event. Modify the Title and add any relevant details that weren't in the template.
Note: you can change defaults from a template on the event.
5. Set the event deadlines: RSVP, question, and submission
6. The event will inherit the suppliers from the project. Add an additional supplier to the event
Note: Introductory emails use your email client and DO sent to supplier emails. Be sure to only use test records.
7. Review the response sheet and evaluation options from the template.
8. Invite stakeholders
9. Preview the event
10. Publish the RFP
11. (Optional) review the confirmation text and click Confirm and Publish RFP to Suppliers

PREREQUISITES:
- User logged in with Sourcing Manager permissions
- Required data available in Workday system
