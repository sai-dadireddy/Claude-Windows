DESCRIPTION:
Verify that all required FLSA calendars are built out in the tenant and the start day is correct

WORKDAY ROLE: Payroll Administrator
EST. DURATION: 5 minutes

PREREQUISITES:
- User logged in with Payroll Administrator permissions
- List of expected FLSA calendars with correct work week start days
- Understanding of FLSA overtime requirements

ELECTRON STEPS:
1. enter search box as "View FLSA Work Period Calendar"
2. wait for search results
3. click search result containing "View FLSA Work Period Calendar"
4. wait for page to load
5. verify page title contains "FLSA Work Period Calendar"
6. verify calendar list displays all configured calendars
7. scroll to view Calendar Name column
8. scroll to view Work Week Start Day column
9. verify all required calendars are present
10. screenshot as "USP-1-0080_calendar_list.png"
11. click first calendar row to view details
12. wait for details to load
13. verify Work Week Start Day field displays
14. verify Effective Date field displays
15. verify Status is Active
16. screenshot as "USP-1-0080_calendar_details.png"
17. click button "Done"
18. enter search box as "All FLSA Work Period Calendars"
19. wait for search results
20. click search result containing "All FLSA Work Period Calendars"
21. wait for report to load
22. verify all calendars display with start days
23. screenshot as "USP-1-0080_complete.png"

VERIFICATION:
- [ ] All required FLSA calendars are present in the tenant
- [ ] Work week start day is correct for each calendar
- [ ] Calendars are Active and properly effective-dated
- [ ] No duplicate calendars for the same population
- [ ] Calendar names follow naming standards
- [ ] Start day aligns with business requirements (typically Sunday or Saturday)
- [ ] No error messages displayed

API ALTERNATIVE:
SOAP:
  WSDL: Payroll
  Operation: Get_FLSA_Work_Period_Calendars

REST:
  Endpoint: /payroll/v2/flsaWorkPeriodCalendars
  Method: GET
