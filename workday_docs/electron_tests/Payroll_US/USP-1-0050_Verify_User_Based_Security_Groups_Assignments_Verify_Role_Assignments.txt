DESCRIPTION:
Verify the correct workers are assigned to Payroll Security Groups (i.e. Payroll Administrator, Payroll Administrator FRA, Payroll Calculations Administrator, Payroll Integrations Administrator, Payroll Partner, Organization Partner (on Pay Groups), Payroll Settlement Specialist and Payroll Auditor)

WORKDAY ROLE: Security Administrator
EST. DURATION: 5 minutes

PREREQUISITES:
- User logged in with Security Administrator permissions
- List of expected users for each security group
- List of expected role assignments by organization

ELECTRON STEPS:
1. enter search box as "View Security Group"
2. wait for search results
3. click search result containing "View Security Group"
4. wait for page to load
5. verify page title contains "Security Group"
6. select security group dropdown as "Payroll Administrator"
7. wait for group details to load
8. click tab "Members"
9. wait for members list to load
10. verify member list displays assigned users
11. scroll to view all members
12. screenshot as "USP-1-0050_payroll_admin.png"
13. select security group dropdown as "Payroll Calculations Administrator"
14. wait for group details to load
15. click tab "Members"
16. verify member list displays
17. screenshot as "USP-1-0050_calc_admin.png"
18. select security group dropdown as "Payroll Integrations Administrator"
19. wait for group details to load
20. click tab "Members"
21. screenshot as "USP-1-0050_integrations_admin.png"
22. select security group dropdown as "Payroll Settlement Specialist"
23. wait for group details to load
24. click tab "Members"
25. screenshot as "USP-1-0050_settlement_specialist.png"
26. select security group dropdown as "Payroll Auditor"
27. wait for group details to load
28. click tab "Members"
29. screenshot as "USP-1-0050_payroll_auditor.png"
30. select security group dropdown as "Payroll Partner"
31. wait for group details to load
32. click tab "Assignable Role"
33. verify role assignments display with organizations
34. scroll to view all role assignments
35. screenshot as "USP-1-0050_payroll_partner_roles.png"
36. select security group dropdown as "Organization Partner"
37. wait for group details to load
38. click tab "Assignable Role"
39. verify pay groups are assigned to partners
40. screenshot as "USP-1-0050_org_partner_roles.png"
41. screenshot as "USP-1-0050_complete.png"

VERIFICATION:
- [ ] Payroll Administrator has correct members assigned
- [ ] Payroll Calculations Administrator has correct members
- [ ] Payroll Integrations Administrator has correct members
- [ ] Payroll Settlement Specialist has correct members
- [ ] Payroll Auditor has correct members (read-only access)
- [ ] Payroll Partner role assigned to correct organizations
- [ ] Organization Partner assigned to all pay groups
- [ ] No unauthorized users in administrator groups
- [ ] Segregation of duties maintained
