DESCRIPTION:
Verify that all expected pay run groups are present in the tenant and that each pay run group contains the expected pay groups

WORKDAY ROLE: Payroll Administrator
EST. DURATION: 5 minutes

PREREQUISITES:
- User logged in with Payroll Administrator permissions
- Access to View Pay Run Group task
- Knowledge of expected pay run group names and configurations

ELECTRON STEPS:
1. enter search box as "View Pay Run Group"
2. wait for search results
3. click search result containing "View Pay Run Group"
4. wait for page to load
5. verify pay run group list is displayed
6. verify column "Pay Run Group Name" is visible
7. verify column "Pay Groups" is visible
8. scroll through list to view all groups
9. screenshot as "USP-5-0040_pay_run_groups.png"
10. click on first pay run group to view details
11. wait for details to load
12. verify included pay groups listed
13. verify processing order displayed
14. verify company assignments are correct
15. screenshot as "USP-5-0040_pay_run_group_detail.png"
16. navigate back to pay run group list
17. verify all expected US pay groups are assigned to a run group
18. verify no pay groups missing from any run group
19. screenshot as "USP-5-0040_complete.png"

VERIFICATION:
- [ ] All expected pay run groups are present in the tenant
- [ ] Each pay run group contains the expected pay groups
- [ ] Processing order is appropriate
- [ ] No gaps in pay group coverage
- [ ] Settlement associations are correct
- [ ] Company assignments are valid

PAY RUN GROUP COMPONENTS TO VERIFY:
- Pay Run Group Name
- Pay Run Group ID
- Included Pay Groups
- Processing Order
- Company assignments

RELATED REPORTS:
- Pay Run Group Summary
- Pay Group Assignment Report
- Payroll Processing Schedule
- Pay Group Configuration Report
