DESCRIPTION:
Verify that all required FLSA calendars are built out in the tenant and the start day is correct

WORKDAY ROLE: Payroll Administrator
EST. DURATION: 5 minutes

PREREQUISITES:
- User logged in with Payroll Administrator permissions
- Access to View FLSA Work Period Calendar task
- Knowledge of expected FLSA calendar configurations

ELECTRON STEPS:
1. enter search box as "View FLSA Work Period Calendar"
2. wait for search results
3. click search result containing "View FLSA Work Period Calendar"
4. wait for page to load
5. verify calendar list displays all configured FLSA calendars
6. verify column "Work Week Start Day" is visible
7. verify column "Effective Date" is visible
8. scroll to view all calendars
9. verify each calendar has correct work week start day
10. verify calendars are active and effective-dated correctly
11. screenshot as "USP-5-0100_flsa_calendars.png"
12. click on first calendar to view details
13. wait for details to load
14. verify work week start day setting (Sunday through Saturday)
15. verify work week start time setting
16. verify effective date is appropriate
17. screenshot as "USP-5-0100_flsa_calendar_detail.png"
18. navigate back to calendar list
19. screenshot as "USP-5-0100_complete.png"

VERIFICATION:
- [ ] All required FLSA calendars are present in tenant
- [ ] Work week start day is correct for each calendar
- [ ] Calendars are active and effective-dated correctly
- [ ] No duplicate calendars for same employee population
- [ ] Calendar names follow naming convention
- [ ] All non-exempt employee groups are covered

FLSA CALENDAR COMPONENTS TO VERIFY:
- Calendar Name
- Work Week Start Day (Sunday through Saturday)
- Work Week Start Time (typically midnight)
- Effective Date
- Status (Active/Inactive)
