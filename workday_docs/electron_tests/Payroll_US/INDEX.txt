================================================================================
FILE NAMING CONVENTION
================================================================================

Format: {Scenario_ID}_{Scenario_Name_sanitized}.txt

Examples:
- USP-1-0010_Review_Period_Schedule_Configuration.txt
- USP-4-0090-05_Retro_Comp_Change_Payroll_Impact.txt
- USP-P1-0110_Prove_1_Map_Validation.txt

Special Characters Removed:
- Parentheses: () → removed
- Slashes: / → removed
- Colons: : → removed
- Commas: , → removed
- Spaces: → replaced with underscores

Length Limit: 100 characters (name truncated if longer)

SEARCH & FILTER TIPS
================================================================================

Find Automated Tests:
  grep -l "CONFIDENCE:" *.txt | wc -l

Find Manual Tests:
  grep -l "STATUS: \[MANUAL REQUIRED\]" *.txt | wc -l

  grep -l "CONFIDENCE: \[HIGH\]" *.txt

  ls USP-1-*.txt        # All Unit tests

  ls USP-P1-*.txt       # All Parallel tests
