DESCRIPTION:
Verify that FLSA calendar rules are configured so that the correct workers will be assigned to the right calendar (and only one calendar per worker)

WORKDAY ROLE: Payroll Administrator
EST. DURATION: 5 minutes

PREREQUISITES:
- User logged in with Payroll Administrator permissions
- Access to View FLSA Work Period Calendar Rules task
- Knowledge of expected FLSA calendar rule configurations

ELECTRON STEPS:
1. enter search box as "View FLSA Work Period Calendar Rules"
2. wait for search results
3. click search result containing "View FLSA Work Period Calendar Rules"
4. wait for page to load
5. verify rules table displays all configured rules
6. verify column "Rule Name" is visible
7. verify column "FLSA Calendar" is visible
8. verify column "Eligibility Criteria" is visible
9. verify column "Priority" is visible
10. scroll to review all rules
11. screenshot as "USP-5-0110_flsa_rules.png"
12. click on first rule to view details
13. wait for details to load
14. verify eligibility criteria is clearly defined
15. verify rule priority prevents conflicts
16. verify associated FLSA calendar assignment
17. screenshot as "USP-5-0110_rule_detail.png"
18. navigate back to rules list
19. verify all non-exempt employee groups have rules
20. verify each worker can match exactly one rule
21. screenshot as "USP-5-0110_complete.png"

VERIFICATION:
- [ ] FLSA calendar rules are correctly configured
- [ ] Correct workers will be assigned to the right calendar
- [ ] Each worker matches exactly one rule (no conflicts)
- [ ] Rules cover all non-exempt employees
- [ ] Rule priority prevents overlapping assignments
- [ ] Eligibility criteria are accurate and complete

FLSA CALENDAR RULE COMPONENTS TO VERIFY:
- Rule Name
- Associated FLSA Calendar
- Eligibility Criteria (worker attributes)
- Priority Order
- Effective Date
- Status (Active/Inactive)

ELIGIBILITY CRITERIA EXAMPLES:
- Pay Group
- Worker Type
- Position
- Location
- Supervisory Organization
- Job Profile

RELATED TASKS:
- View FLSA Work Period Calendar
- Create FLSA Work Period Calendar Rules
- Edit FLSA Work Period Calendar Rules
- Assign FLSA Calendar to Workers

RELATED REPORTS:
- FLSA Work Period Calendar Assignments
- Workers by FLSA Calendar Rule
- FLSA Calendar Rule Audit
- Workers Missing FLSA Calendar Assignment
