DESCRIPTION:
For all US payroll companies verify that the following are in place: 1.Fiscal Schedule 2. Account Set 3. Account Posting Rule 4. Ledger Year Usage Count should not be zero 5. Company Transaction Tax ID 6. Is Transaction Tax ID is 'Yes' 7. Is Primary Tax ID is 'Yes' 8. Bank Accounts 9. Bank Routing Rules 10. Costing Allocations (optional)

WORKDAY ROLE: Payroll Administrator / Company Administrator
EST. DURATION: 5 minutes

PREREQUISITES:
- User logged in with Payroll Administrator / Company Administrator permissions
- Access to Company Accounting Details report
- Access to company configuration tasks

ELECTRON STEPS:
1. enter search box as "View Company Details"
2. wait for search results
3. click search result containing "View Company Details"
4. wait for page to load
5. select US payroll company from company prompt
6. wait for company details to load
7. verify Fiscal Schedule is assigned and valid
8. screenshot as "USP-5-0120_fiscal_schedule.png"
9. scroll to Accounting section
10. verify Account Set is configured
11. verify Account Posting Rule is assigned
12. verify Ledger Year Usage Count is not zero
13. screenshot as "USP-5-0120_accounting.png"
14. scroll to Tax ID section
15. verify Company Transaction Tax ID is present
16. verify "Is Transaction Tax ID" is set to "Yes"
17. verify "Is Primary Tax ID" is set to "Yes"
18. screenshot as "USP-5-0120_tax_id.png"
19. scroll to Banking section
20. verify Bank Accounts are configured
21. verify Bank Routing Rules are in place
22. screenshot as "USP-5-0120_banking.png"
23. scroll to Costing section
24. verify Costing Allocations if configured (optional)
25. screenshot as "USP-5-0120_complete.png"

VERIFICATION:
- [ ] Fiscal Schedule is assigned to company
- [ ] Account Set is configured for payroll posting
- [ ] Account Posting Rule is correctly assigned
- [ ] Ledger Year Usage Count is greater than zero
- [ ] Company Transaction Tax ID is present
- [ ] "Is Transaction Tax ID" is "Yes"
- [ ] "Is Primary Tax ID" is "Yes"
- [ ] Bank Accounts are configured
- [ ] Bank Routing Rules are in place
- [ ] Costing Allocations configured (if applicable)

CONFIGURATION COMPONENTS TO VERIFY:
1. Fiscal Schedule - defines accounting periods
2. Account Set - links payroll codes to GL accounts
3. Account Posting Rule - determines posting behavior
4. Ledger Year - ensures GL periods are available
5. Transaction Tax ID - required for tax filing
6. Primary Tax ID - must be "Yes" for reporting
7. Bank Accounts - for payroll settlement
8. Bank Routing Rules - determines fund routing
9. Costing Allocations - cost center assignments (optional)

- View Payroll Accounting Results

- Edit Company Details

RELATED REPORTS:

- Payroll Accounting Summary
- Bank Account Configuration Report
- GL Account Mapping Report
