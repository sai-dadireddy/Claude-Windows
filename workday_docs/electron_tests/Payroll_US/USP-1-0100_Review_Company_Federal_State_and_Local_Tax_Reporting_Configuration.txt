TEST ID: USP-1-0100
FUNCTIONAL AREA: Payroll for the United States
TEST NAME: Review Company Federal, State and Local Tax Reporting Configuration
CONFIDENCE: [HIGH] Score: 8.5/10
SOURCE: [KB Article: kb_payroll_tax_reporting.txt]
================================================================================

DESCRIPTION:
Verify that all expected company tax configuration (federal, state and local) is present in the tenant and that all overrides are correct: 1. Federal: Tax ID, Supplemental Tax Method, Exempt Status, Employer Type 2. State and Local: Tax IDs, SUI Rates, Supplemental Tax Methods, Exempt Status

WORKDAY ROLE: Payroll Administrator
EST. DURATION: 15 minutes

PREREQUISITES:
- User logged in with Payroll Administrator permissions
- List of expected company tax configuration settings
- Current SUI rates for each state
- List of required local tax authorities

ELECTRON STEPS:
1. enter search box as "Company US Tax Reporting"
2. wait for search results
3. click search result containing "Company US Tax Reporting"
4. wait for page to load
5. verify page title contains "Company US Tax Reporting"
6. select company dropdown as "{Company_Name}"
7. wait for tax data to load
8. verify Federal Tax Settings section displays
9. verify Federal EIN field displays value in XX-XXXXXXX format
10. verify Supplemental Tax Method field displays (Flat or Aggregate)
11. verify Federal Exempt Status field displays
12. verify Employer Type field displays
13. screenshot as "USP-1-0100_federal_tax.png"
14. click tab "State"
15. wait for state tax list to load
16. verify state tax table displays all operating states
17. scroll to view State Tax ID column
18. scroll to view SUI Rate column
19. scroll to view Supplemental Tax Method column
20. scroll to view State Exempt Status column
21. verify SUI rates are current year rates
22. screenshot as "USP-1-0100_state_tax.png"
23. click tab "Local"
24. wait for local tax list to load
25. verify local tax authorities display
26. verify local tax IDs are configured
27. scroll to view all local jurisdictions
28. screenshot as "USP-1-0100_local_tax.png"
29. click related actions menu
30. click link "View Details"
31. wait for details page to load
32. verify all tax overrides are documented
33. screenshot as "USP-1-0100_complete.png"

VERIFICATION:
- [ ] Federal EIN is correct (9 digits: XX-XXXXXXX format)
- [ ] Supplemental tax method is appropriate for company
- [ ] Federal exempt status is accurate if applicable
- [ ] Employer type is correct (501(c)(3), Government, etc.)
- [ ] All operating states are configured
- [ ] State tax IDs are valid and current
- [ ] SUI rates are current (update annually)
- [ ] State supplemental methods are correct
- [ ] State exemptions are documented
- [ ] All applicable local taxes configured
- [ ] Local tax IDs are correct
- [ ] Local withholding rates are current
- [ ] Reciprocity settings are accurate
- [ ] No error messages displayed

  WSDL: Payroll

  Endpoint: /payroll/v2/companyUSTaxReporting
