DESCRIPTION:
Verify that FLSA calendar rules are configured so that the correct workers will be assigned to the right calendar (and only one calendar per worker)

WORKDAY ROLE: Payroll Administrator
EST. DURATION: 5 minutes

PREREQUISITES:
- User logged in with Payroll Administrator permissions
- List of expected FLSA calendar assignment rules
- Understanding of worker eligibility criteria for each calendar

ELECTRON STEPS:
1. enter search box as "View FLSA Work Period Calendar Rules"
2. wait for search results
3. click search result containing "View FLSA Work Period Calendar Rules"
4. wait for page to load
5. verify page title contains "FLSA Work Period Calendar Rules"
6. verify rules table displays all configured rules
7. scroll to view Rule Name column
8. scroll to view Priority column
9. scroll to view Eligibility Criteria column
10. scroll to view FLSA Calendar column
11. verify rules are ordered by priority
12. screenshot as "USP-1-0090_rules_list.png"
13. click first rule row to view details
14. wait for details to load
15. verify Eligibility Criteria section displays
16. verify worker attributes defined for assignment
17. screenshot as "USP-1-0090_rule_details.png"
18. click button "Done"
19. enter search box as "Test FLSA Work Period Calendar Assignment"
20. wait for search results
21. click search result containing "Test FLSA Work Period Calendar Assignment"
22. wait for page to load
23. enter worker field as "{test_worker_id}"
24. click button "OK"
25. wait for results to display
26. verify assigned calendar displays
27. verify only one calendar is assigned
28. screenshot as "USP-1-0090_complete.png"

VERIFICATION:
- [ ] FLSA calendar rules cover all non-exempt employees
- [ ] Each worker matches exactly one rule (no conflicts)
- [ ] Rule priority order prevents duplicate assignments
- [ ] Eligibility criteria are accurate and complete
- [ ] Rules cover all job profiles/locations as needed
- [ ] No workers fall through without calendar assignment
- [ ] Priority order is logical and documented
- [ ] No error messages displayed

  Operation: Get_FLSA_Work_Period_Calendar_Rules

  Endpoint: /payroll/v2/flsaWorkPeriodCalendarRules
