TEST ID: USP-1-0010
FUNCTIONAL AREA: Payroll for the United States
TEST NAME: Review Period Schedule Configuration
CONFIDENCE: [HIGH] Score: 8.5/10
SOURCE: [KB Article: kb_payroll_period_schedule.txt]
================================================================================

DESCRIPTION:
Verify that all expected period schedules are present in the tenant, dates are correct (period start, period end, payment date, and Accounting Date Override) and that forward accrual settings are as expected. Verify that period schedules are linked to Absence / Time Tracking as applicable. Ensure that that the correct number of periods are in place for each year. If applicable, add Continuous Payroll Calculation configuration.

WORKDAY ROLE: Payroll Administrator
EST. DURATION: 5 minutes

PREREQUISITES:
- User logged in with Payroll Administrator permissions
- Access to View Period Schedule task
- List of expected period schedules for verification

ELECTRON STEPS:
1. enter search box as "View Period Schedule"
2. wait for search results
3. click search result containing "View Period Schedule"
4. wait for page to load
5. verify page title contains "Period Schedule"
6. select period schedule dropdown as "{Biweekly Schedule}"
7. wait for periods to load
8. verify period table displays all periods
9. scroll to view Period Start Date column
10. scroll to view Period End Date column
11. scroll to view Payment Date column
12. scroll to view Accounting Date Override column
13. verify period dates are consecutive without gaps
14. screenshot as "USP-1-0010_period_dates.png"
15. click related actions menu
16. click link "View Details"
17. wait for details page to load
18. verify Forward Accrual Settings section displays
19. verify Continuous Payroll Calculation section displays
20. screenshot as "USP-1-0010_forward_accrual.png"
21. click button "Done"
22. enter search box as "Maintain Period Schedule"
23. wait for search results
24. click search result containing "Maintain Period Schedule"
25. wait for page to load
26. verify period count matches expected periods per year
27. screenshot as "USP-1-0010_complete.png"

VERIFICATION:
- [ ] All expected period schedules are present
- [ ] Period start and end dates are correct
- [ ] Payment dates fall on valid business days
- [ ] Accounting Date Override configured where needed
- [ ] Forward accrual settings match requirements
- [ ] Correct number of periods per year (52 weekly, 26 biweekly, 24 semimonthly, 12 monthly)
- [ ] Absence/Time Tracking links configured if applicable
- [ ] Continuous Payroll Calculation configured if applicable
- [ ] No error messages displayed

API ALTERNATIVE:
SOAP:
  WSDL: Payroll
  Operation: Get_Period_Schedules

REST:
  Endpoint: /payroll/v2/periodSchedules
  Method: GET
