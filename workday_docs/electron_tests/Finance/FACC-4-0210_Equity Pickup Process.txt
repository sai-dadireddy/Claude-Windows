DESCRIPTION:
Run an equity pickup process to record parent company's share of
investee's net income under the equity method of accounting. This applies
when parent has significant influence (typically 20-50% ownership) but
not control over the investee.

PREREQUISITES:
- User logged in with Controller or Finance Manager role
- Investment relationship defined with equity method
- Ownership percentage configured
- Investee net income recorded
- Target accounting period open

ELECTRON STEPS:
1. enter search box as "Run Equity Pickup"
2. wait for search results
3. click search result containing "Run Equity Pickup"
4. wait for page to load
5. select Parent Company as "{parent_company}"
6. select Investee as "{investee_company}"
7. enter Accounting Date as "{current_period_date}"
8. select Period as "{fiscal_period}"
9. verify Ownership Percentage shows "{ownership_percentage}"
10. click button "Calculate"
11. wait for calculation results
12. verify Equity Pickup Amount calculated correctly
13. verify Investment Account to be updated
14. click button "Submit"
15. wait for confirmation
16. verify message contains "submitted"
17. screenshot as "FACC-4-0210_complete.png"

VERIFICATION:
- [ ] Equity pickup calculation completed successfully
- [ ] Ownership percentage applied correctly
- [ ] Investment account debited
- [ ] Equity in earnings account credited
- [ ] Entries balanced
- [ ] No error messages displayed

API ALTERNATIVE:
- SOAP: Submit_Equity_Pickup (Financial_Management WSDL)
