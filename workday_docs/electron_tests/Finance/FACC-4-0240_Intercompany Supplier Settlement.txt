DESCRIPTION:
Perform intercompany supplier settlement to clear intercompany receivable
and payable balances between affiliated companies. This creates netting
entries to settle amounts due to/from related entities and generates
settlement documentation.

PREREQUISITES:
- User logged in with Finance Manager or Controller role
- Intercompany transactions posted between affiliates
- Matching intercompany receivables and payables exist
- Settlement rules configured
- Target accounting period open

ELECTRON STEPS:
1. enter search box as "Create Intercompany Settlement"
2. wait for search results
3. click search result containing "Create Intercompany Settlement"
4. wait for page to load
5. select Primary Company as "{primary_company}"
6. select Settlement Type as "Supplier Settlement"
7. enter Settlement Date as "{settlement_date}"
8. select Affiliate Company as "{affiliate_company}"
9. click button "Search" to find open balances
10. wait for search results
11. verify open intercompany amounts displayed
12. select transactions to settle
13. verify net settlement amount calculated
14. click button "Preview Settlement"
15. verify settlement entries are correct
16. click button "Submit"
17. wait for confirmation
18. verify message contains "settlement" and "submitted"
19. screenshot as "FACC-4-0240_complete.png"

VERIFICATION:
- [ ] Intercompany settlement created successfully
- [ ] Open balances identified and matched
- [ ] Netting entries generated correctly
- [ ] Intercompany accounts cleared
- [ ] Settlement balanced
- [ ] No error messages displayed

API ALTERNATIVE:
- SOAP: Submit_Intercompany_Settlement (Financial_Management WSDL)
- Get_Intercompany_Open_Balances to retrieve unsettled amounts
