DESCRIPTION:
Create a recurring journal template for repetitive accounting entries such as
monthly depreciation, rent allocations, or standard accruals. Templates can
be used to generate journals automatically or on-demand each period.

PREREQUISITES:
- User logged in with Accountant or Finance Manager role
- Recurring journal template does not already exist
- Ledger accounts and worktags configured
- Target company has recurring journal functionality enabled

ELECTRON STEPS:
1. enter search box as "Create Recurring Journal Template"
2. wait for search results
3. click search result containing "Create Recurring Journal Template"
4. wait for page to load
5. enter Template Name as "Monthly Depreciation - FACC-4-0180"
6. select Company as "{test_company}"
7. select Ledger Type as "Actuals"
8. select Journal Source as "Recurring"
9. select Frequency as "Monthly"
10. enter Start Date as "{start_date}"
11. enter End Date as "{end_date}"
12. enter Memo Template as "Monthly depreciation entry"
13. click button "Add"
14. wait for template line section
15. click link "Add Template Line"
16. enter Ledger Account as "{depreciation_expense}"
17. enter Debit Amount as "{monthly_depreciation}"
18. select Cost Center as "{cost_center}"
19. enter Line Memo as "Depreciation expense"
20. click button "OK"
21. click link "Add Template Line"
22. enter Ledger Account as "{accumulated_depreciation}"
23. enter Credit Amount as "{monthly_depreciation}"
24. select Cost Center as "{cost_center}"
25. enter Line Memo as "Accumulated depreciation"
26. click button "OK"
27. verify template shows "In Balance"
28. click button "Submit"
29. wait for confirmation
30. verify message contains "submitted" or "created"
31. screenshot as "FACC-4-0180_complete.png"

VERIFICATION:
- [ ] Recurring journal template created successfully
- [ ] Template balanced (debits = credits)
- [ ] Frequency and date range correct
- [ ] Template active and available for use
- [ ] No error messages displayed

API ALTERNATIVE:
- SOAP: Put_Recurring_Journal_Template (Financial_Management WSDL)
- Get_Recurring_Journal_Templates to retrieve existing templates
