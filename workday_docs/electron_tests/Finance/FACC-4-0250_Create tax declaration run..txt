DESCRIPTION:
Create and run a tax declaration to prepare tax reporting data for filing.
This process aggregates taxable transactions, calculates tax liabilities,
and generates output for regulatory submission (VAT returns, sales tax, etc.).

PREREQUISITES:
- User logged in with Tax Administrator or Finance Manager role
- All taxable transactions posted for the period
- Tax categories and rates configured
- Tax declaration definition set up
- Target tax period open

ELECTRON STEPS:
1. enter search box as "Create Tax Declaration Run"
2. wait for search results
3. click search result containing "Create Tax Declaration Run"
4. wait for page to load
5. select Company as "{test_company}"
6. select Tax Authority as "{tax_authority}"
7. select Declaration Type as "{declaration_type}"
8. enter From Date as "{period_start}"
9. enter To Date as "{period_end}"
10. click button "Preview"
11. wait for preview results
12. verify taxable transactions included
13. verify tax amounts calculated correctly
14. verify tax categories applied
15. click button "Submit"
16. wait for confirmation
17. verify message contains "declaration" and "submitted"
18. verify declaration status shows "Created" or "Pending"
19. screenshot as "FACC-4-0250_complete.png"

VERIFICATION:
- [ ] Tax declaration run created successfully
- [ ] All taxable transactions included
- [ ] Tax calculations correct
- [ ] Tax liability amounts accurate
- [ ] Declaration ready for filing
- [ ] No error messages displayed

API ALTERNATIVE:
- SOAP: Submit_Tax_Declaration (Financial_Management WSDL)
- Get_Tax_Declaration_Results for retrieval
