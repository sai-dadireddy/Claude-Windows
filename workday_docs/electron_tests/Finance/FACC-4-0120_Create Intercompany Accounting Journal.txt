DESCRIPTION:
Create an intercompany accounting journal to record transactions between
two or more legal entities. Intercompany journals automatically create
reciprocal entries in affiliated companies and must balance across all entities.

PREREQUISITES:
- User logged in with Accountant or Finance Manager role
- Multiple companies with intercompany relationships configured
- Target companies have open accounting periods
- Intercompany accounts set up in chart of accounts

ELECTRON STEPS:
1. enter search box as "Create Intercompany Accounting Journal"
2. wait for search results
3. click search result containing "Create Intercompany Accounting Journal"
4. wait for page to load
5. select Primary Company as "{primary_company}"
6. enter Accounting Date as "{current_period_date}"
7. select Ledger Type as "Actuals"
8. select Journal Source as "Intercompany"
9. enter Memo as "Intercompany transaction - FACC-4-0120"
10. click button "Add"
11. wait for journal line section
12. click link "Add Journal Line"
13. select Company as "{primary_company}"
14. enter Ledger Account as "{intercompany_receivable}"
15. enter Debit Amount as "{transaction_amount}"
16. select Intercompany Affiliate as "{affiliate_company}"
17. select Cost Center as "{cost_center}"
18. enter Line Memo as "Due from affiliate"
19. click button "OK"
20. click link "Add Journal Line"
21. select Company as "{affiliate_company}"
22. enter Ledger Account as "{intercompany_payable}"
23. enter Credit Amount as "{transaction_amount}"
24. select Intercompany Affiliate as "{primary_company}"
25. select Cost Center as "{affiliate_cost_center}"
26. enter Line Memo as "Due to primary company"
27. click button "OK"
28. verify journal shows "In Balance"
29. verify intercompany balancing is complete
30. click button "Submit"
31. wait for confirmation
32. verify message contains "submitted"
33. screenshot as "FACC-4-0120_complete.png"

VERIFICATION:
- [ ] Intercompany journal created successfully
- [ ] Reciprocal entries balanced across entities
- [ ] Intercompany affiliates properly linked
- [ ] Journal routed for approval
- [ ] No error messages displayed

API ALTERNATIVE:
- SOAP: Submit_Accounting_Journal (Financial_Management WSDL)
- Intercompany entries require Intercompany_Affiliate_Reference
