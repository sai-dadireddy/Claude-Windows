DESCRIPTION:
Run a revaluation process to adjust foreign currency denominated
balance sheet accounts to current exchange rates. This creates
unrealized gain/loss entries for period-end reporting in the
company's base currency.

PREREQUISITES:
- User logged in with Finance Manager or Controller role
- Foreign currency balances exist in target accounts
- Current exchange rates entered for revaluation date
- Revaluation gain/loss accounts configured
- Target accounting period open

ELECTRON STEPS:
1. enter search box as "Run Revaluation"
2. wait for search results
3. click search result containing "Run Revaluation"
4. wait for page to load
5. select Company as "{test_company}"
6. enter Revaluation Date as "{period_end_date}"
7. select Currency as "{foreign_currency}"
8. select Revaluation Rule as "{revaluation_rule}"
9. select Account Type as "Balance Sheet" or "All"
10. click button "Preview"
11. wait for preview results
12. verify accounts to be revalued listed
13. verify exchange rate difference calculated
14. verify unrealized gain/loss amounts
15. click button "Submit"
16. wait for confirmation
17. verify message contains "revaluation" and "submitted"
18. screenshot as "FACC-4-0220_complete.png"

VERIFICATION:
- [ ] Revaluation process completed successfully
- [ ] Current exchange rates applied
- [ ] Unrealized gain/loss calculated correctly
- [ ] Revaluation journal entries created
- [ ] Entries balanced
- [ ] No error messages displayed

API ALTERNATIVE:
- SOAP: Submit_Revaluation (Financial_Management WSDL)
- Get_Currency_Rate_Types for exchange rate information
