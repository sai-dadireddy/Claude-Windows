DESCRIPTION:
Create an accounting adjustment journal to correct errors or reclassify
transactions in the general ledger. Adjustments are used to fix posting
errors, reallocate expenses, or make period-end corrections.

PREREQUISITES:
- User logged in with Accountant or Finance Manager role
- Original transaction identified for adjustment
- Target company has open accounting period
- Appropriate adjustment reason codes configured

ELECTRON STEPS:
1. enter search box as "Create Accounting Journal"
2. wait for search results
3. click search result containing "Create Accounting Journal"
4. wait for page to load
5. select Company as "{test_company}"
6. enter Accounting Date as "{current_period_date}"
7. select Ledger Type as "Actuals"
8. select Journal Source as "Adjustment"
9. enter Memo as "Accounting adjustment - FACC-4-0150 - Correcting {original_ref}"
10. click button "Add"
11. wait for journal line section
12. click link "Add Journal Line"
13. enter Ledger Account as "{original_account}"
14. enter Credit Amount as "{adjustment_amount}"
15. select Cost Center as "{original_cost_center}"
16. enter Line Memo as "Reverse original posting"
17. click button "OK"
18. click link "Add Journal Line"
19. enter Ledger Account as "{correct_account}"
20. enter Debit Amount as "{adjustment_amount}"
21. select Cost Center as "{correct_cost_center}"
22. enter Line Memo as "Correct posting to proper account"
23. click button "OK"
24. verify journal shows "In Balance"
25. click tab "Attachments"
26. click button "Add Attachment"
27. upload supporting documentation
28. click button "Submit"
29. wait for confirmation
30. verify message contains "submitted"
31. screenshot as "FACC-4-0150_complete.png"

VERIFICATION:
- [ ] Adjustment journal created successfully
- [ ] Original transaction reference documented
- [ ] Debits equal credits (balanced)
- [ ] Supporting documentation attached
- [ ] Journal routed for approval
- [ ] No error messages displayed

API ALTERNATIVE:
- SOAP: Submit_Accounting_Journal (Financial_Management WSDL)
- Adjustments use Journal_Source of "Adjustment" type
