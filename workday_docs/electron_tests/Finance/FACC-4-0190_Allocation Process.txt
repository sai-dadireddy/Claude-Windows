DESCRIPTION:
Run an allocation process to distribute costs from one or more source
cost centers or accounts to target cost centers based on predefined
allocation rules and statistical drivers (headcount, square footage, etc.).

PREREQUISITES:
- User logged in with Finance Manager or Controller role
- Allocation rules defined and active
- Statistics or allocation bases populated for the period
- Source transactions posted to GL
- Target accounting period open

ELECTRON STEPS:
1. enter search box as "Run Allocation"
2. wait for search results
3. click search result containing "Run Allocation"
4. wait for page to load
5. select Company as "{test_company}"
6. select Allocation Rule as "{allocation_rule_name}"
7. enter Accounting Date as "{current_period_date}"
8. select Ledger Type as "Actuals"
9. enter Period as "{fiscal_period}"
10. click button "Preview" to validate allocation
11. wait for preview results
12. verify allocation amounts are correct
13. verify source and target cost centers match
14. click button "Submit"
15. wait for confirmation
16. verify message contains "allocation" and "submitted"
17. screenshot as "FACC-4-0190_complete.png"

- [ ] Allocation process completed successfully

- [ ] Resulting journals created and balanced
