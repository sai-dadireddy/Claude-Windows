DESCRIPTION:
Create a manual accounting journal entry to record transactions in the general
ledger. Journal entries are used for accruals, adjustments, reclassifications,
depreciation, and corrections not captured through operational transactions.

PREREQUISITES:
- User logged in with Accountant or Finance Manager role
- Target company has open accounting period
- Ledger accounts and worktags configured

ELECTRON STEPS:
1. enter search box as "Create Accounting Journal"
2. wait for search results
3. click search result containing "Create Accounting Journal"
4. wait for page to load
5. select Company as "{test_company}"
6. enter Accounting Date as "{current_period_date}"
7. select Ledger Type as "Actuals"
8. select Journal Source as "Manual"
9. enter Memo as "Test journal entry - FACC-4-0110"
10. click button "Add"
11. wait for journal line section
12. click link "Add Journal Line"
13. enter Ledger Account as "{debit_account}"
14. enter Debit Amount as "{debit_amount}"
15. select Cost Center as "{cost_center}"
16. enter Line Memo as "Debit entry line"
17. click button "OK"
18. click link "Add Journal Line"
19. enter Ledger Account as "{credit_account}"
20. enter Credit Amount as "{credit_amount}"
21. select Cost Center as "{cost_center}"
22. enter Line Memo as "Credit entry line"
23. click button "OK"
24. verify journal shows "In Balance"
25. click button "Submit"
26. wait for confirmation
27. verify message contains "submitted"
28. screenshot as "FACC-4-0110_complete.png"

VERIFICATION:
- [ ] Journal entry created successfully
- [ ] Debits equal credits (balanced)
- [ ] Journal routed for approval
- [ ] No error messages displayed

API ALTERNATIVE:
- SOAP: Submit_Accounting_Journal (Financial_Management WSDL)
- Request includes: Company, Accounting_Date, Ledger_Type, Journal_Lines
