DESCRIPTION:
Create an intercompany journal for service charges between affiliated
companies. This records management fees, shared service charges, or
cost allocations between related legal entities with proper IC elimination.

PREREQUISITES:
- User logged in with Accountant or Finance Manager role
- Intercompany service charge agreement in place
- Intercompany accounts configured
- Target companies have open accounting periods

ELECTRON STEPS:
1. enter search box as "Create Intercompany Accounting Journal"
2. wait for search results
3. click search result containing "Create Intercompany Accounting Journal"
4. wait for page to load
5. select Primary Company as "{service_provider}"
6. enter Accounting Date as "{current_period_date}"
7. select Ledger Type as "Actuals"
8. select Journal Source as "Intercompany Service Charge"
9. enter Memo as "IC Service Charge - FACC-4-0120-01"
10. click button "Add"
11. wait for journal line section
12. click link "Add Journal Line"
13. select Company as "{service_provider}"
14. enter Ledger Account as "{service_revenue}"
15. enter Credit Amount as "{service_fee}"
16. select Intercompany Affiliate as "{service_receiver}"
17. select Cost Center as "{provider_cost_center}"
18. enter Line Memo as "Management fee income"
19. click button "OK"
20. click link "Add Journal Line"
21. select Company as "{service_receiver}"
22. enter Ledger Account as "{service_expense}"
23. enter Debit Amount as "{service_fee}"
24. select Intercompany Affiliate as "{service_provider}"
25. select Cost Center as "{receiver_cost_center}"
26. enter Line Memo as "Management fee expense"
27. click button "OK"
28. verify journal shows "In Balance"
29. verify intercompany elimination entries will be created
30. click button "Submit"
31. wait for confirmation
32. verify message contains "submitted"
33. screenshot as "FACC-4-0120-01_complete.png"

VERIFICATION:
- [ ] Intercompany service charge journal created
- [ ] Revenue/expense properly recorded in each company
- [ ] Intercompany affiliates properly linked
- [ ] Entries will eliminate on consolidation
- [ ] Journal routed for approval
- [ ] No error messages displayed

API ALTERNATIVE:
- SOAP: Submit_Accounting_Journal (Financial_Management WSDL)
- Intercompany service charges require proper affiliate references
