# Quick Start Guide

## Get Started in 5 Minutes!

### Prerequisites Check
- [ ] Python 3.9+ installed
- [ ] pip package manager
- [ ] Text editor (VS Code, etc.)
- [ ] Web browser (Chrome, Firefox, Edge)

## Step-by-Step Setup

### 1. Install MkDocs Material
```bash
pip install mkdocs-material
```

**Verify**:
```bash
mkdocs --version
# Should show: mkdocs, version X.X.X
```

### 2. Navigate to Project
```bash
cd "C:\Users\SainathreddyDadiredd\OneDrive - ERPA\Claude\docs\learning\mkdocs-interactive"
```

### 3. Start Development Server
```bash
mkdocs serve
```

**Expected Output**:
```
INFO - Building documentation...
INFO - Cleaning site directory
INFO - Documentation built in 0.52 seconds
INFO - Serving on http://127.0.0.1:8000/
```

### 4. Open in Browser
Go to: **http://127.0.0.1:8000**

## Common Issues

### Issue: "mkdocs: command not found"
**Fix**:
```bash
pip install --upgrade mkdocs-material
# On Windows, may need: python -m pip install mkdocs-material
```

### Issue: Progress not saving
**Check**: Not in private/incognito mode?
**Fix**: Use normal browser window

### Issue: Page shows 404
**Check**: Is the navigation correct?
**Fix**: Some pages may not exist yet (create stub)
