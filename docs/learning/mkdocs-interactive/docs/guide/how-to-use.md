## Progress Tracking Details

### Checkboxes
Each topic page has a checkbox at the top. Click it to mark complete.

**What Gets Saved**:
- Completion status (true/false)
- Completion timestamp
- Topic ID

**Where**: Browser localStorage (your device only)

### Notes
Each topic has a notes section at the bottom.

**How to Use**:
- Type your notes in the textarea
- Click away (blur event) to auto-save
- Notes persist across browser sessions

**Tips**:
- Summarize key concepts in your own words
- Save code snippets that worked
- Note questions to research later
- Link to external resources

### Dashboard
View your progress at any time:

**Stats Shown**:
- Overall progress percentage
- Days learning (since start)
- Topics completed this week
- Current streak (consecutive days)
- Progress by part (12 parts)
- Recently completed topics (last 5)

**Actions**:
- Export progress (JSON download)
- Reset all progress (careful!)

## Using AI Providers

### GPT/Codex (OpenAI)
**Best For**: Production code, debugging, optimization

> "Implement multi-head attention in PyTorch with detailed comments."

### Gemini (Google)
**Best For**: Search quality, scale patterns, evaluation

**Example**:
> "How does Google handle vector search at billion-doc scale?"

**Provider Icons**: Look for these on each topic page to see which AI excels.

## Troubleshooting

### Progress Not Saving?
**Check**: Not in private/incognito mode?
**Fix**: Use normal browsing mode

### Notes Disappearing?
**Check**: Did you clear browser data?
**Fix**: Export regularly as backup

### Page Not Loading?
**Check**: Check browser console for errors
**Fix**: Hard refresh (Ctrl+Shift+R)

## FAQ

**Q: Can I use this offline?**
A: Yes! After first load, content is cached.

**Q: Can I print topics?**
A: Yes, use browser print (Ctrl+P).

**Q: Can I customize colors?**
A: Yes, edit `mkdocs.yml` and rebuild.

**Q: Can multiple people use this?**
A: Yes, but each needs their own browser profile.

**Q: How do I backup progress?**
A: Dashboard ‚Üí "Export Progress" button.
