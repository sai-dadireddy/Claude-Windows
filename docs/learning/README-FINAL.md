## Troubleshooting

### Progress Not Saving?
**Check**:
1. Not in private/incognito mode?
2. Browser allows localStorage?
3. Check console for JavaScript errors

**Fix**: Use normal browsing mode, enable localStorage in browser settings

### Site Not Building?
**Check**:
```bash
# Verify MkDocs installed
mkdocs --version

# Reinstall if needed
pip install --upgrade mkdocs-material
```

### 404 on GitHub Pages?
**Check**:
1. Repository Settings â†’ Pages â†’ Source = `gh-pages` branch
2. Wait 2-3 minutes after deployment
3. Verify `index.html` exists in `gh-pages` branch
